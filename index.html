Welcome!

CSS Gallery is a demonstration of the full power of CSS. This interactive photo gallery is built entirely from CSS (compiled from LESS code); there is no Javascript anywhere.

You can select a category from the fully-functional hamburger menu to filter the photos; you can switch between list and thumbnail views using the bar on top; you can click a photo to open a lightbox scaled to your browser window; you can even navigate between photos from within the lightbox.

To get started, select an option from the hamburger menu in the upper right corner, or navigate directly to All.

To see how it works, see the source on GitHub.
